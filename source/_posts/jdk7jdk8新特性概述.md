在oracle停止对jdk6更新，jdk8发布之后，公司终于要把生产环境更新到jdk7，下面列一下jdk7,8的可能需要关注的新特性。
## jdk7
* G1垃圾回收
* fork-join框架
* 二进制变量
* Switch支持String类型
* try-with-resource
* 简化泛型
* catch多个异常
* InvokeDynamic
* 更多NIO的API

## jdk8
* lambda
* Nashorn JavaScript Engine
* JDBC 4.2
* Bulk Data Operations for Collections
* Base64
* 新的时间函数
* HTTP URL Permissions
* Repeating Annotations
